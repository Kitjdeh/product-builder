# Project Blueprint

## Overview

This project is an interactive calendar application. It allows users to view dates, navigate between months and years, and select specific dates.

## Style, Design, and Features

### Initial Version

*   **Core Functionality:**
    *   Displays a monthly calendar grid.
    *   Allows navigation to previous/next months and years.
    *   Highlights the current date ("today").
    *   Allows users to select a date, which is highlighted.
    *   Includes a "Today" button to jump back to the current month.
    *   Features a modal window for quick year selection.
*   **Design and Layout:**
    *   A clean, centered calendar component with a gradient background.
    *   Responsive design that adapts to different screen sizes.
    *   Modern aesthetics with rounded corners, shadows, and smooth transitions.
    *   Clear visual hierarchy for the header, weekdays, and days.
*   **Technology:**
    *   Built with standard HTML, CSS, and JavaScript.
    *   No external frameworks or libraries are used.
